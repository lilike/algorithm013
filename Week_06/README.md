做动态规划的题目,关键是如何抽象出做题方法很重要.

动态规划题目的做题步骤一般是:

1. 首先使用暴力,递归回溯贪心的方法去做一遍
2. 使用带缓存的递归回溯做一遍
3. 进行动态规划优化,主要是要记录在递归和回溯过程中的最优解.
    可以使用一维数组,二维数组,或是两个变量进行
4. 写出DP方程,DP方程最开始是猜想,然后如果时间够的话,可以以数学方法进行证明
5. 根据DP方程去写出代码


总之,动态规划类的题目,后期我还需要持续的做题练习.
